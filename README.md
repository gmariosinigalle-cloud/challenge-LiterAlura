📚 LiterAlura

LiterAlura es una aplicación de consola desarrollada con Java y Spring Boot que consume la API pública de Gutendex para buscar libros, almacenarlos en una base de datos PostgreSQL y permitir su consulta mediante distintos criterios.

La aplicación permite:

🔎 Buscar libros por título y registrarlos en la base de datos

📖 Listar libros registrados

👨‍💼 Listar autores registrados

📅 Consultar autores vivos en un año determinado

🌎 Filtrar libros por idioma

El proyecto implementa:

Consumo de API REST externa

Conversión de datos JSON a objetos Java

Persistencia con Spring Data JPA

Relaciones ManyToMany entre libros y autores

Consultas derivadas en repositorios

Es un proyecto orientado a consolidar conocimientos en backend con Spring Boot, JPA y modelado relacional.
